# Idle Content Matrix

## 1. Content rhythm

Every ten Exploration Levels adds a pool bracket. Every five levels previews
the next addition. Levels 25, 50, 75, and 100 add a build decision or ceremony.

| Level | Content beat |
|---:|---|
| 1 | Node begins with three recognizable resources |
| 5 | Preview next resource silhouette |
| 10 | Bracket II and first natural event family |
| 15 | First node-type Perk |
| 20 | Bracket III and Node Commission variety |
| 25 | First specialization choice |
| 30 | Bracket IV and trade-oriented resources |
| 35 | Second node-type Perk |
| 40 | Bracket V and dimension-support resources |
| 50 | Bracket VI, rare caps, second specialization |
| 60 | Bracket VII, favored-resource control, third type Perk |
| 70 | Bracket VIII and Deep Expedition |
| 75 | Mastery challenge unlock |
| 80 | Bracket IX and reliability improvements |
| 85 | Final node-type Perk |
| 90 | Bracket X and capped mastery resources |
| 100 | Capstone, façade, frontier eligibility |

## 2. Node identities

| Node | Primary fantasy | Strength | Constraint |
|---|---|---|---|
| Mining | Dig deeper into valuable strata | Metals and enchanting resources | Valuable drops are capped |
| Farming | Build a broad renewable supply | Highest volume and order flexibility | Lower individual item value |
| Woodcutting | Collect building palettes | Most controllable common output | Fewer high-rarity vanilla drops |
| Livestock | Run ranches and fisheries | Diverse crafting/food utility | Several outputs require expeditions |
| Hunter | Send teams after dangerous mobs | Best active-event rewards | Lowest passive rate |

## 8. Event families

Events must differ mechanically, not only by name.

| Family | Main stat | Duration | Main reward |
|---|---|---:|---|
| Survey | Stamina | Short | Exploration EXP |
| Resource Run | Diligence | Medium | High quantity |
| Rare Discovery | Luck | Medium | Rare pool |
| Rescue | Speed | Short and urgent | Chronicle/relationship reward |
| Endurance | Stamina + team size | Long | Worker EXP |
| Deep Expedition | Balanced team | Long | Capstone/mastery materials |

Team result:

`TeamScore = Σ(role-adjusted stats) × synergy × preparation`

Luck shifts grade odds but never creates a fail state. Speed reduces duration.
Stamina increases Node/Worker EXP. Diligence increases quantity.

## 11. Content authoring rule

Every new resource must define:

- Node family and unlock level.
- Passive/event eligibility.
- Initial and mature weight.
- Daily/weekly cap if valuable.
- At least one current use.
- Journal clue and discovery text.
- Commission eligibility.
- Future MMORPG relationship, if any.

No item should enter a pool solely because it is rare.
